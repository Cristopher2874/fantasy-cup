# Explanation And Reference Material

This folder keeps planning notes and external samples out of the runnable app root.

- `game_explanation.md`: tournament rules and product explanation.
- `app-dev-plan.md`: phased build plan and future OCI notes.
- `app-system-diagrams.md`: detailed architecture and workflow diagrams for team alignment.
- `system-overview.md`: one-page overview with a single full-system diagram.
- `main-modules-simple.md`: simple six-module explanation for introducing the idea.
- `samples/`: original reference scripts for API-Sports, hosted containers, shell tool use, skill references, and vector stores.
